# FundRaise AI 🚀

A modern AI-powered crowdfunding platform built with:

- Next.js 15
- FastAPI
- PostgreSQL
- Tailwind CSS
- Razorpay
- JWT Authentication

## Features

- User Authentication
- Campaign Creation
- Secure Donations
- Admin Dashboard
- AI Campaign Writer
- Analytics
- Responsive UI

## Tech Stack

Frontend:
- Next.js
- TypeScript
- Tailwind CSS

Backend:
- FastAPI
- SQLAlchemy
- PostgreSQL
